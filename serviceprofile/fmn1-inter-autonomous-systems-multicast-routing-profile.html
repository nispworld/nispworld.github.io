Title
Inter-Autonomous Systems Multicast Routing Service Profile 1.0
Description
The Inter-Autonomous Systems Multicast Routing Profile provides standards and guidance for multicast routing between inter-autonomous systems.

Standards

Obligation: Mandatory, Lifecycle: Current

The following standards shall apply for all IP interconnections

  • Protocol Independent Multicast - Sparse Mode (PIM-SM): Protocol Specification (Revised) (IETF RFC 4601:2006)
  • Host Extensions for IP Multicasting (IETF RFC 1112:1989)
  • Internet Group Management Protocol, Version 3 (IETF RFC 3376:2002)

Obligation: Mandatory, Lifecycle: Current

MNEs, as well as MNXs with their own multicast capability, shall provide a Rendezvous Point (RP) supporting the following IP multicast protocol standards

  • Multicast Source Discovery Protocol (MSDP) (IETF RFC 3618:2003)
  • Multiprotocol Extensions for BGP-4 (IETF RFC 4760:2007)

Obligation: Mandatory, Lifecycle: Current

The following standards shall apply to multicast routing

  • The Internet Multicast Address Allocation Architecture (IETF RFC 2908:2000)
  • IANA Guidelines for IPv4 Multicast Address Assignments (IETF RFC 3171:2001)
  • Administratively Scoped IP Multicast (IETF RFC 2365:1998)
