FMN Spiral 3.0 Informal Messaging Profile
Description
The Informal Messaging Profile provides standards and guidance for SMTP settings and the marking of informal messages.

Standards

Obligation: MANDATORY, Lifecycle: CURRENT

Regarding Simple Mail Transfer Protocol (SMTP), the following standards are mandated for interoperability of e-mail services within the Mission Network.

  • Simple Mail Transfer Protocol (RFC 5321:2008)
  • SMTP Service Extension for Message Size Declaration (RFC 1870:1995)
  • SMTP Service Extension for Returning Enhanced Error Codes (RFC 2034:1996)
  • SMTP Service Extension for Command Pipelining (RFC 2920:2000)
  • MTP Service Extension for Secure SMTP over Transport Layer Security (TLS) (RFC 3207:2002)
  • Simple Mail Transfer Protocol (SMTP) Service Extension for Delivery Status Notifications (DSNs) (RFC 3461:2003)
  • SMTP Service Extension for Authentication (RFC 4954:2007)
