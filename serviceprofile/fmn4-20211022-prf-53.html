Title
ISR Library Interface Profile
Description
The ISR Library Interface is the standard interface for querying and accessing heterogeneous product libraries maintained by various nations.

Standards

Obligation: MANDATORY, Lifecycle: CURRENT

The Basic Image Interchange Format (BIIF) is mandated for interoperability of ISR libraries.

  • Image Processing and Interchange (IPI) - Functional Specification - Part 5: Basic Image Interchange Format (BIIF) (12087-5:1998)
  • Technical Corrigendum 1 to International Standard ISO/IEC 12087-5:1998 (12087-5 Cor 1:2001)
  • Technical Corrigendum 2 to International Standard ISO/IEC 12087-5:1998 (12087-5 Cor 2:2002)

Obligation: MANDATORY, Lifecycle: CURRENT

The following NATO standards provide the specification as well as business rules for interoperability of ISR libraries.

  • NATO Standard ISR Library Interface (AEDP-17 Ed A Ver 1:2018)

Obligation: MANDATORY, Lifecycle: CURRENT

Implementation of STANAG 5525 in the context of the ISR Library Interface Profile is limited to the definition of unique keys that could be used to unambiguously refer to an external information object that is modelled in accordance with STANAG 5525. Note that AEDP-17 refers to the metadata attribute “JC3IEDMIdentifier” on page G-15, but to “identifierJC3IEDM” on page G-79. The correct attribute to use is “identifierJC3IEDM”.

  • Joint C3 Information Exchange Data Model Baseline 3.1.4 (JC3IEDM Baseline 3.1.4:2012)

Obligation: MANDATORY, Lifecycle: CURRENT

The following NATO standards are mandated for interoperability of ISR library products.

  • US Motion Imagery Standards Board (MISB) - Motion Imagimary Standards Profile-2015.1 (NNSTD MISP-2015.1:2016)
  • NATO Secondary Imagery Format (NSIF) STANAG 4545 Implementation Guide (AEDP-04 Ed 2 Ver 1:2013)
  • NATO Ground Moving Target Indicator (GMTI) Format STANAG 4607 Implementation Guide (AEDP-07 Ed 2 Ver 1:2010)

Obligation: MANDATORY, Lifecycle: CURRENT

The following international standards are mandated for interoperability of ISR libraries.

  • Representation of Names of Languages Part 2: Alpha-3 (639-2:1998)
  • Information technology -- Metadata registries (MDR) -- Part 3: Registry metamodel and basic attributes (11179-3:2013)
  • Information technology -- Open Distributed Processing -- Interface Definition Language (14750:1999)
