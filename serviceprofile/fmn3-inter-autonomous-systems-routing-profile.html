Title
FMN Spiral 3.0 Inter-Autonomous Systems Routing Profile
Description
The Inter-Autonomous Systems Routing Profile provides standards and guidance for routing between inter-autonomous systems.

Standards

Obligation: MANDATORY, Lifecycle: CURRENT

The following standards apply for all IP interconnections.

  • Border Gateway Protocol (BGP) Communities Attribute (RFC 1997:1996)
  • BGP Extended Communities Attribute (RFC 4360:2006)
  • Capabilities Advertisement with BGP-4 (RFC 5492:2009)
  • A Border Gateway Protocol 4 (BGP-4) (RFC 4271:2006)
  • Multiprotocol Extensions for BGP-4 (RFC 4760:2007)
  • Revised Error Handling for BGP UPDATE Messages (RFC 7606:2015)
  • BGP Support for Four-Octet Autonomous System (AS) Number Space (RFC 6793:2012)
  • Autonomous-System-Wide Unique BGP Identifier for BGP-4 (RFC 6286:2011)
  • IANA Registries for BGP Extended Communities (RFC 7153:2014)

Obligation: MANDATORY, Lifecycle: CURRENT

The following standard applies for unicast routing.

  • Classless Inter-domain Routing (CIDR): The Internet Address Assignment and Aggregation Plan (RFC 4632:2006)

Obligation: MANDATORY, Lifecycle: CURRENT

The following standard is added to improve MD5-based BGP-authentication

  • The Generalized TTL Security Mechanism (GTSM) (RFC 5082:2007)

Obligation: CONDITIONAL, Lifecycle: CURRENT

The following standard can be added to improve MD5-based BGP-authentication, depending on bilateral agreement.

  • BGP Operations and Security (RFC 7454:2015)

Obligation: RECOMMENDED, Lifecycle: CURRENT

Additionally, the following standard applies for 32-bit autonomous system numbers (ASN).

  • 4-Octet AS Specific BGP Extended Community (RFC 5668:2009)
