Title
IPSec-based Media Infrastructure Security Profile
Description
The IPSec-based Media Infrastructure Security Profile provides security standards that are used for security of media infrastructure based on Internet Protocol Security (IPSec).

Standards

Obligation: Conditional, Lifecycle: Current

Securing the MN Media infrastructure can be done in several ways and that the selection of the appropriate method is to be done during the mission planning. For this specific method, the following standard apply.

  • Internet Key Exchange Protocol Version 2 (IKEv2) (IETF RFC 7296:2014)
  • IP Encapsulating Security Payload (ESP) (IETF RFC 4303:2005)
  • IKE and IKEv2 Authentication Using the Elliptic Curve Digital Signature Algorithm (ECDSA) (IETF RFC 4754:2007)
  • Elliptic Curve Groups modulo a Prime (ECP Groups) for IKE and IKEv2 (IETF RFC 5903:2010)
  • Signature Authentication in the Internet Key Exchange Version 2 (IKEv2) (IETF RFC 7427:2015)
  • Generic Raw Public-Key Support for IKEv2 (IETF RFC 7670:2016)
