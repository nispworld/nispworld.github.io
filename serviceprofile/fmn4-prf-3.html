Title
FMN Spiral 4.0 Text-based Collaboration Profile
Description
The Text-based Collaboration Profile provides standards and guidance to establish a basic near-real time text-based group collaboration capability (chat) for time critical reporting and decision making in military operations.

Standards

Obligation: Mandatory, Lifecycle: Current

The following standards are the base IETF protocols for interoperability of chat services.

  • Extensible Messaging and Presence Protocol (XMPP): Core (IETF RFC 6120:2011)
  • Extensible Messaging and Presence Protocol (XMPP): Instant Messaging and Presence (IETF RFC 6121:2011)
  • Extensible Messaging and Presence Protocol (XMPP): Address Format (IETF RFC 6122:2011)

Obligation: Mandatory, Lifecycle: Current

The following standards are required to achieve compliance for an XMPP Server and an XMPP Client dependent upon the categorisation of presenting a core or advanced instant messaging service interface.

  • XEP-0012: Last Activity (XMPP XEP-0012:2008)
  • Service Discovery (XMPP XEP-0030:2017)
  • XEP-0047: In-Band Bytestreams (XMPP XEP-0047:2012)
  • XEP-0054: vcard-temp (XMPP XEP-0054:2008)
  • XEP-0055: Jabber Search (XMPP XEP-0055:2009)
  • Publish-Subscribe (XMPP XEP-0060:2020)
  • XEP-0092: Software Version (XMPP XEP-0092:2007)
  • JID Escaping (XMPP :2007)
  • XEP-0114: Jabber Component Protocol (XMPP XEP-0114:2012)
  • Entity Capabilities (XMPP XEP-0115:2020)
  • XEP-0160: Best Practices for Handling Offline Messages (XMPP XEP-0160:2016)
  • XEP-0199: XMPP Ping (XMPP XEP-0199:2009)
  • XEP-0202: Entity Time (XMPP XEP-0202:2009)
  • XEP-0203: Delayed Delivery (XMPP XEP-0203:2009)
  • Server Dialback (XMPP XEP-0220:2015)
