Title
Proposed FMN Spiral 4 Video-based Collaboration Serviceprofile
Description
The Video-based Collaboration Profile provides standards and guidance for the implementation and configuration of Video Tele Conferencing (VTC) systems and services in a federated mission network.

Standards

Obligation: Conditional, Lifecycle: Candidate

Not required at this time, but when available it can be implemented between dedicated network segments after approval from the MN administrative authority.

  • The Binary Floor Control Protocol (BFCP) (IETF RFC 4582:2006)

Obligation: Mandatory, Lifecycle: Candidate

The following standards are required for audio coding in VTC.

  • Low-complexity coding at 24 and 32 kbit/s for hands-free operation in systems with low frame loss (ITU-T G.722.1 (2005) Corrigendum 1 (06/08):2008)
  • Pulse code modulation (PCM) of voice frequencies (ITU-T G.711:1988)

Obligation: Mandatory, Lifecycle: Candidate

The following standards are required for video coding in VTC.

  • Advanced video coding for generic audiovisual services (ITU-T H.264:2017)
  • RTP Payload Format for H.264 Video (IETF RFC 6184:2011)
