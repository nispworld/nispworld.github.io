Title
GeoPackage Profile
Description
GeoPackage is an open standard developed and maintained by the Open Geospatial Consortium (OGC). It provides for a single container to hold all geospatial data types (vector, raster, and elevation) and is able to support not only the physical transfer of geospatial data but also to directly publish the data as services. It is light-weight enough that it can be employed in mobile devices requiring geospatial data content and is able to support geospatial data exchange in a Degraded/Denied, Disrupted, Intermittent, and Limited (bandwidth) (DDIL) environment. OGC GeoPackage is supported by a wide variety of commercial and open source software applications.
