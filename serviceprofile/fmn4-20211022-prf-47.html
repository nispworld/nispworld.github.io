Title
Geospatial Web Feeds Profile
Description
The Geospatial Web Feeds Profile provides standards and guidance for in support of Geospatial Services to deliver geospatial content to web sites and to user agents, including the encoding of location as part of web feeds.

GML subset for point "gml Point", line "gml LineString", polygon "gml Polygon", and box "gml Envelope". In Atom feeds, location shall be specified using Atom 1.0's official extension mechanism in combination with the GeoRSS GML Profile 1.0 whereby a "georss where" element is added as a child of the element.
