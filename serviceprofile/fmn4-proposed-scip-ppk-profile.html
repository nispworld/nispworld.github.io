Title
Proposed FMN Spiral 4 SCIP PPK Serviceprofile
Description
In the context of secure communications, PPK is the Pre-Placed Key, which is a symmetric encryption key, pre-positioned in a cryptographic unit.

Standards

Obligation: Mandatory, Lifecycle: Candidate

When PPK is applied for the Secure Communications Interoperability Protocol (SCIP), the following standards need to be followed.

  • NATO Pre Placed Key (PPK) Key Material Format and Fill Checks Specification Rev.1.0 (CIS3 C&IP SCIP-233.104:)
  • NATO Point-to-Point and Multipoint PPK-Processing Specification Rev.1.0 (CIS3 C&IP SCIP-233.304:)
  • Interoperable Terminal Priority (TP) Community of Interest (COI) Specification rev.1.0 (CIS3 C&IP SCIP-233.350:2012)
  • Application State Vector Processing Specification rev.1.2 (CIS3 C&IP SCIP-233.401:2012)
  • NATO Fixed Filler Generation Specification Rev. 1.0. (CIS3 C&IP SCIP-233.422:)
  • Point-to-Point Cryptographic Verification Specification Rev. 1.1. (CIS3 C&IP SCIP-233.441:)
  • AES-256 Encryption Algorithm Specification Rev. 1.0. (CIS3 C&IP SCIP-233.601:2011)
