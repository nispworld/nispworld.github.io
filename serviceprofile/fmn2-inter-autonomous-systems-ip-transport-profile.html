Title
Inter-Autonomous Systems IP Transport Service Profile 2.0
Description
The Inter-Autonomous Systems IP Transport Profile provides standards and guidance for Edge Transport Services between autonomous systems, using Internet Protocol (IP) over point-to-point Ethernet links on optical fibre.

Standards

Obligation: Mandatory, Lifecycle: Current

The use of LC-connectors is required for network interconnections inside shelters (or inside other conditioned infrastructure).

  • Characteristics of a single-mode optical fibre and cable (ITU-T G.652:2016)
  • Interface standard for LC connectors with protective housings related to IEC 61076-3-106 (IEC 61754-20-100:2012)

Obligation: Mandatory, Lifecycle: Current

  • Generic cabling for customer premises (ISO/IEC 11801:2002)

Obligation: Optional, Lifecycle: Current

If the interconnection point is outside a shelter in a harsh environment, the interconnection shall follow STANAG 4290 or MIL-DTL-83526 connector specifications.

  • Connectors, Fiber Optic, Circular, Environmental Resistant, Hermaphroditic, General Specification for. D (MIL-STD DTL 83526:2006)
  • Standard for Gateway Multichannel Cable Link (Optical) (NATO STANAG 4290 Ed 1:2015)

Obligation: Mandatory, Lifecycle: Current

Section 3 - Clause 38 - 1000BASE-LX, nominal transmit wavelength 1310nm.

  • Single-mode fiber using 1,310 nm wavelength (IEEE 802.3-2012:2012)

Obligation: Mandatory, Lifecycle: Current
