Title
Inter-Autonomous Systems Routing Profile
Description
The Inter-Autonomous Systems Routing Profile provides standards and guidance for routing between inter-autonomous systems. The best current practice for the Border Gateway Protocol (BGP) based network routing operations and security is described in RFC 7454 - "BGP Operations and Security". Deployment guidance with regards to the application of BGP in the Internet is described in IETF RFC 1772 1995.

Standards

Obligation: MANDATORY, Lifecycle: CANDIDATE

The following standards apply for all IP interconnections.

  • A Border Gateway Protocol 4 (BGP-4) (RFC 4271:2006)
  • Multiprotocol Extensions for BGP-4 (RFC 4760:2007)
  • Capabilities Advertisement with BGP-4 (RFC 5492:2009)
  • Autonomous-System-Wide Unique BGP Identifier for BGP-4 (RFC 6286:2011)
  • BGP Support for Four-Octet Autonomous System (AS) Number Space (RFC 6793:2012)
  • Revised Error Handling for BGP UPDATE Messages (RFC 7606:2015)
  • Default External BGP (EBGP) Route Propagation Behavior without Policies (RFC 8212:2017)

Obligation: CONDITIONAL, Lifecycle: CANDIDATE

Additionally, the following standards apply for use of communities, extended communities and 32-bit extended communities for traffic engineering purposes.

  • Border Gateway Protocol (BGP) Communities Attribute (RFC 1997:1996)
  • BGP Extended Communities Attribute (RFC 4360:2006)
  • 4-Octet AS Specific BGP Extended Community (RFC 5668:2009)
  • IANA Registries for BGP Extended Communities (RFC 7153:2014)
  • Policy Behavior for Well-Known BGP Communities (RFC 8642:2019)

Obligation: MANDATORY, Lifecycle: CANDIDATE

The following standard is added to improve security of BGP peering

  • The Generalized TTL Security Mechanism (GTSM) (RFC 5082:2007)

Obligation: MANDATORY, Lifecycle: CANDIDATE

The following standards are added to improve BGP resilience through faster detection of network failures

  • Bidirectional Forwarding Detection (BFD) (RFC 5880:2010)
  • Bidirectional Forwarding Detection (BFD) for IPv4 and IPv6 (Single Hop) (RFC 5881:2010)
  • Bidirectional Forwarding Detection (BFD) for Multihop Paths (RFC 5883:2010)
