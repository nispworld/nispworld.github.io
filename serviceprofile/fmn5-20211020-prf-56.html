Title
Informal Messaging Profile
Description
The Informal Messaging Profile provides standards and guidance for settings of Simple Mail Transfer Protocol (SMTP).

Standards

Obligation: Mandatory, Lifecycle: Candidate

These standards are mandated for interoperability of e-mail services within the mission network.

  • SMTP Service Extension for Message Size Declaration (IETF RFC 1870:1995)
  • SMTP Service Extension for Returning Enhanced Error Codes (IETF RFC 2034:1996)
  • SMTP Service Extension for Command Pipelining (IETF RFC 2920:2000)
  • MTP Service Extension for Secure SMTP over Transport Layer Security (TLS) (IETF RFC 3207:2002)
  • Simple Mail Transfer Protocol (SMTP) Service Extension for Delivery Status Notifications (DSNs) (IETF RFC 3461:2003)
  • SMTP Service Extension for Authentication (IETF RFC 4954:2007)
  • Simple Mail Transfer Protocol (IETF RFC 5321:2008)
  • Internet Message Format (IETF RFC 5322:2008)
