Obligation: Mandatory, Lifecycle: Current

The following international standards are mandated for interoperability of ISR libraries.

  • Representation of Names of Languages Part 2: Alpha-3 (ISO 639-2:1998)
  • Information technology -- Metadata registries (MDR) -- Part 3: Registry metamodel and basic attributes (ISO/IEC 11179-3:2013)
  • Image Processing and Interchange (IPI) - Functional Specification - Part 5: Basic Image Interchange Format (BIIF) (ISO/IEC 12087-5:1998)
  • Information technology -- Open Distributed Processing -- Interface Definition Language (ISO/IEC 14750:1999)
