FMN Spiral 3.0 ISR Library Interface Profile
Description
The ISR Library Interface is the standard interface for querying and accessing heterogeneous product libraries maintained by various nations.

Standards

Obligation: MANDATORY, Lifecycle: CURRENT

The following international standards are mandated for interoperability of ISR libraries.

  • Representation of Names of Languages Part 2: Alpha-3 (639-2:1998)
  • Information technology -- Metadata registries (MDR) -- Part 3: Registry metamodel and basic attributes (11179-3:2013)
  • Image Processing and Interchange (IPI) - Functional Specification - Part 5: Basic Image Interchange Format (BIIF) (12087-5:1998)
  • Information technology -- Open Distributed Processing -- Interface Definition Language (14750:1999)

Obligation: MANDATORY, Lifecycle: CURRENT

The following NATO standards are mandated for interoperability of ISR libraries. (For STANAG 4559 Ed 4 Only Standard AEDP-17 Ed. A Ver. 1 NATO Standard ISR Library Interface.)

  • NATO Secondary Imagery Format (NSIF) STANAG 4545 Implementation Guide (AEDP-04 Ed 2 Ver 1:2013)
  • NATO Ground Moving Target Indicator (GMTI) Format STANAG 4607 Implementation Guide (AEDP-07 Ed 2 Ver 1:2010)
  • NATO Standard ISR Library Interface (AEDP-17 Ed A Ver 1:2018)
  • NATO Standard ISR Streaming Services (AEDP-18 Ed A Ver 1:2018)
  • NATO Standard ISR Workflow Architecture (AEDP-19 Ed A Ver 1:2018)
  • US Motion Imagery Standards Board (MISB) - Motion Imagimary Standards Profile-2015.1 (NNSTD MISP-2015.1:2016)

Obligation: MANDATORY, Lifecycle: CURRENT

Note implementation of STANAG 5525 in the context of the ISR Library Interface Profile is limited to the definition of unique keys that could be used to unambiguously refer to an external information object that is modelled in accordance with STANAG 5525.

  • Joint C3 Information Exchange Data Model (JC3IEDM) (STANAG 5525 Ed 1:2007)
