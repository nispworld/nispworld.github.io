Title
Proposed FMN Spiral 4 Inter-Autonomous Systems Routing Serviceprofile
Description
The Inter-Autonomous Systems Routing Profile provides standards and guidance for routing between inter-autonomous systems.

Standards

Obligation: Mandatory, Lifecycle: Candidate

The following standard is added to improve MD5-based BGP-authentication.

  • The Generalized TTL Security Mechanism (GTSM) (IETF RFC 5082:2007)

Obligation: Recommended, Lifecycle: Candidate

Additionally, the following standard applies for 32-bit autonomous system numbers (ASN).

  • 4-Octet AS Specific BGP Extended Community (IETF RFC 5668:2009)

Obligation: Mandatory, Lifecycle: Candidate

The following standard applies for unicast routing.

  • Classless Inter-domain Routing (CIDR): The Internet Address Assignment and Aggregation Plan (IETF RFC 4632:2006)

Obligation: Mandatory, Lifecycle: Candidate

The following standards apply for all IP interconnections.

  • BGP Communities Attribute (IETF RFC 1997:1996)
  • BGP Extended Communities Attribute (IETF RFC 4360:2006)
  • Capabilities Advertisement with BGP-4 (IETF RFC 5492:2009)
  • Border Gateway Protocol 4 (BGP-4) (IETF RFC 4271:2006)
  • Multiprotocol Extensions for BGP-4 (IETF RFC 4760:2007)
  • Revised Error Handling for BGP UPDATE Messages (IETF RFC 7606:2015)
  • BGP Support for Four-Octet Autonomous System (AS) Number Space (IETF RFC 6793:2012)
  • Autonomous-System-Wide Unique BGP Identifier for BGP-4 (IETF RFC 6286:2011)
  • IANA Registries for BGP Extended Communities (IETF RFC 7153:2014)

Obligation: Conditional, Lifecycle: Candidate

The following standard can be added to improve MD5-based BGP-authentication, depending on bilateral agreement.

  • BGP Operations and Security (IETF RFC 7454:2015)
