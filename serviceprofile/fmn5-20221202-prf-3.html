Title
Text-based Collaboration Core Profile
Description
The Text-based Collaboration Core Profile provides standards and guidance to establish a basic near-real time text-based group collaboration capability (chat) for time critical reporting and decision making in military operations.

Standards

Obligation: MANDATORY, Lifecycle: CANDIDATE

The following standards are the base IETF protocols for interoperability of chat services.

  • Extensible Messaging and Presence Protocol (XMPP): Core (RFC 6120:2011)
  • Extensible Messaging and Presence Protocol (XMPP): Instant Messaging and Presence (RFC 6121:2011)
  • Extensible Messaging and Presence Protocol (XMPP): Address Format (RFC 6122:2011)

Obligation: MANDATORY, Lifecycle: CANDIDATE

The following standards are required to achieve compliance for an XMPP Server and an XMPP Client dependent upon the categorisation of presenting a core or advanced instant messaging service interface.

  • XEP-0045: Multi-User Chat (2019/05) (XEP-0045 (2019/05):2019)
  • XEP-0012: Last Activity (XEP-0012:2008)
  • XEP-0054: vcard-temp (XEP-0054:2008)
  • XEP-0106: JID Escaping (XEP-0106:2007)
  • XEP-0115: Entity Capabilities (2020/05) (XEP-0115 (2020/05):2020)
  • XEP-0160: Best Practices for Handling Offline Messages (2016/01) (XEP-0160 (2016/01):2016)
  • XEP-0199: XMPP Ping (2009/06) (XEP-0199 (2009/06):2009)
  • XEP-0202: Entity Time (XEP-0202:2009)
  • XEP-0203: Delayed Delivery (XEP-0203:2009)
  • XEP-0220: Server Dialback (2015/03) (XEP-0220 (2015/03):2015)
