Title
XML Linking Language (XLink) Version 1.1

Applicability

This specification defines the XML Linking Language (XLink) Version 1.1, which allows elements to be inserted into XML documents in order to create and describe links between resources. It uses XML syntax to create structures that can describe links similar to the simple unidirectional hyperlinks of todayâ€™s HTML, as well as more sophisticated links.
