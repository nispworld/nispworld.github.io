Title
DGIWIG GeoPackage Profile

Applicability

This document is a profile of OGC 12-128r18, OGC GeoPackage Encoding Standard, Version 1.3.1, dated 2021-09-20. It defines specific Defence requirements, recommendations and guidelines for interoperability between producers and consumers of geospatial content in the GeoPackage file format for use by DGIWG member countries.
