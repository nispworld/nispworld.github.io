Title
Resource ReSerVation Protocol (RSVP) -- Version 1 Functional Specification

Applicability

This document describes version 1 of RSVP, a resource reservation setup protocol designed for an integrated services Internet. RSVP provides receiver-initiated setup of resource reservations for multicast or unicast data flows, with good scaling and robustness properties.
