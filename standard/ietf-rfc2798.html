Title
Definition of the inetOrgPerson LDAP Object Class

Applicability

While the X.500 standards define many useful attribute types and object classes, they do not define a person object class that meets the requirements found in todayâ€™s Internet and Intranet directory service deployments. We define a new object class called inetOrgPerson for use in LDAP and X.500 directory services that extends the X.521 standard organizationalPerson class to meet these needs.
