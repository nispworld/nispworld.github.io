Title
Keyboard Layouts Part 3: Complementary layouts of the alphanumeric zone of the alphanumeric section

Applicability

ISO/IEC 9995-3 2010 defines the allocation on a keyboard of a set of graphic characters which, when used in combination with an existing national version keyboard layout or the omplementary Latin group layout as defined in ISO/IEC 9995-3 2010, allows the input of a minimum character repertoire as defined by collection 281 (MES-1) specified in ISO/IEC 10646 and proposes extensions of this minimum repertoire.
