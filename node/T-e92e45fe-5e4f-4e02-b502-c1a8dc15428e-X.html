Title
Remote Desktop Application
Description

The Remote Desktop Application enables users to run a personal computer’s desktop environment remotely on one system (usually a PC, but the concept applies equally to a server), while being displayed on a separate client device. The application has varying features. Some allow attaching to an existing user’s session (i.e. a running desktop) and ‘remote controlling ‘, either displaying the remote control session or blanking the screen. Taking over a desktop remotely is a form of remote administration.
