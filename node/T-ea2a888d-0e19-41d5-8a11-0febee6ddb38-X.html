Title
Maritime Vessel Management Application
Description

The Maritime Vessel Management Application allows the user to manage Vessels, which represent maritime crafts. All naval units, merchant vessels, fishing vessels, pleasure crafts and government ships like coastguard, police, customs and research are stored with the types as defined in APP-20.
