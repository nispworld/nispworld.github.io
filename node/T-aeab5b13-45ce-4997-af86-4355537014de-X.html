Title
ASC Develop Airspace Control Order Process
Description
The ASC Develop Airspace Control Order outlines the development of the [[Airspace Control Order ACO]]. The JACC will promulgate the activation of pre-planned [[Airspace Control Means (ACM) Airspace Control Means (ACMs)]] and/or procedures, and other airspace control information by issuing an ACO. ACMs and/or procedures will be valid for the period of the ACO or the specified time frame. ACO planning is usually synchronized with the ATO planning cycle.
