Title
Maritime Data Source Simulator Application
Description

The Maritime Data Source Simulator Application allows the user to control the data source simulators to be used in the Training Environment. Track Simulator can receive Simulated Object Data from the Ground Truth when it is configured to run in the Training Environment.
