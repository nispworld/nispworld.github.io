Title
Plan Test
Description

The ‘Plan Test ‘ process addresses the pre-test analysis of the evaluation objectives to determine the types and quantities of data needed, the results expected or anticipated from the tests, and the analytical tools needed to conduct the tests and evaluations. The use of validated models and simulation systems during pre-test analysis can aid in determining how to design test scenarios, how to set up the test environment, how to properly instrument the test, how to staff and control test resources, how best to sequence the test trials, and how to estimate outcomes. Test managers determine what valid data exist in historical files that can be applied and what new data must be developed through testing.
