Title
Radio Exploitation Application
Description

The Radio Exploitation Application enables users to analyze and exploit the characteristics of radio frequency signals in order to classify and identify the source of the signal. In particular, the application allows the user to compare new data with records of previously analyzed radio frequency signals. Radio frequency MASINT data are an extension of standard ELINT data.
