Title
Signals Exploitation Application
Description

The Signals Exploitation Application enables users to import, manipulate, analyse, correlate, and visualise geo-referenced SIGINT sensor data in order to support the Operational Task ‘Provide Operational Intelligence, Surveillance, and Reconnaissance ‘. The SIGINT data can be presented as graphics, text overlays, and descriptive text segments as well as geo-referenced overlays on digital maps.
