Title
Maritime Picture Display Application
Description

The Maritime Picture Display Application provides two views in a browser. Application View (AppView) provides the user interface for launching all maritime applications. Geospatial View (GeoView) provides the display of geospatial information on a map. All C4ISR Objects, both military and non-military, are displayed on the Map Panel with predefined symbols when they are sent by the AppView.
