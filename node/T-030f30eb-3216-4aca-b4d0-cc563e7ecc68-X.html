Title
Project Planning Services
Description
The Project Planning Services typically provide the following capabilities across the enterprise or federation project planning, resource assignment, project accounting, project collaboration and project tracking, integrating information for other Support services and systems like Workforce Management Systems and Accounting Systems. Web-based Project Management Appplications and tools typically model and enforce best practices that facilitate reliable and consistent project planning, launch and delivery across the enterprise or federation.
