Title
Maritime Track Analysis Application
Description

Maritime Track Analysis Application performs automatic or manual analysis for detecting unexpected behaviours using the information in Track and Vessel Databases. Live data or off-line data can be used for analysis. A user can perform an analysis on databases within a Maritime Operation. If necessary, a user can import data from an archive into a Maritime Operation which is created for analysis purposes only. The user can then visualise the objects under analysis in the Geospatial View.
