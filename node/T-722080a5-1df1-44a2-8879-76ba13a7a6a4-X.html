Title
Situational Awareness Applications
Description

The Situational Awareness Applications enable users to gain knowledge, cognition and anticipation of events, factors and variables affecting the safe, expedient and effective conduct of missions. Situation Awarenes is the human perception of available elements of data, information, and knowledge; the result of a complex interaction between prior knowledge and beliefs with current perceptions of reality. The Situational Awarenes Applications also enable users to describe the engagement space (operational environment) as a system and identify key subsystems or system elements as well as leverage points using network analysis techniques and conduct what-if analysis in order to support the Knowledge Development function of the Comprehensive Operations Planning Process.
