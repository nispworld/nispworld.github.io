Title
Workflow Application
Description

The Workflow Application enables users to define and manage workflow definitions, and to initiate and monitor workflow instances. A workflow models a repeatable process as a progression of connected activities. The workflow model consists of the following concepts, a start point - initiation of the workflow; an end point - completion of the workflow; the activities - the smallest step or steps within a workflow; decision points - a decision point indicates the flow (linear or branching) of work from one activity to the next activity or to the end point; and the connections between the activities.
