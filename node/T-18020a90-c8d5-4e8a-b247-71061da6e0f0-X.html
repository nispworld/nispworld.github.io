Title
Maritime PIM Management Application
Description

Position and Intended Movement (PIM) is a navigational plot where the intended route and timeline of a unit is illustrated. PIM Routes (a.k.a. tracks) can be used for normal ship/battle group movement (navigation planning), possible threat movements, and for mission planning for aircraft and submarines. The Maritime PIM Application performs PIM management using waypoints, legs and routes.
