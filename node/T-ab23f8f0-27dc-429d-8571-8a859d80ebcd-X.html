Title
Maritime Conversion Application
Description

The Maritime Conversion Application converts given values from a selected unit to another unit in an area. The areas include Distance (Nautical Mile, Statute Mile, Data Mile, Yard, Feet, Metre, Kilometre), Wind speed and force (e.g. Beaufort to km/h), Speed (knots, Data Mile per hour, km/h, m/s, mph), Weight/Mass, Length, Area, Volume, Temperature, Density/Pressure, and position.
