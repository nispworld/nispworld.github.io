Title
Element Management Application
Description

The Element Management Application enables users to managing the physical resources/elements (e.g. servers, storage, routers, security devices incl. crypto devices, etc.) required to provide services to the customer. Element Management includes Performance Management; Fault Management; Configuration Management; Accounting Management; and Security Management.
