Title
Biometric Acquisition Application
Description

The Biometric Acquisition Application enables users to collate multimodal biometric samples from a person and its enrollment into a core biometric database in order to provide an authoritative identification of individuals. The application can retrieve biometric data from the database and perform match of new samples with data from the database. A search for previous enrollment is intrinsic to the enrollment process in order to immediately identify whether the person is black or grey listed.
