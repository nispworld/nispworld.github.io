Title
CIS Security Governance Process
Description

The CIS Security Governance Process enables to establish strategic directions for CIS Security, establish policies to secure CIS and the information it handles, inlcuding standard approaches, minimum and desirable security requirements, definition of metrics, definition of information protection and release policies, definition of CIS Security metadata, and finally, to audit CIS Security for the purpose of accountability.
