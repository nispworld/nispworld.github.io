Title
Manpack Tactical Radio
Description

A Manpack Tactical Radio (MTR) is user equipment that is lightweight, rugged, waterproof. This equipment provides reliable, highly portable voice and data communications. It has a minimum setup and teardown time, operates for many days on a single battery charge and satisfies a need for extended distance communications.
