Title
Trusted Platform Module
Description

A Trusted Platform Module (TPM) is a standard-based chip that is built into computing devices to enable the secure generation of crypto keys and to limit the use of them (to signing/verification or encryption/decryption). TPMs serve as a secure container for crypto key storage and can safeguard other data deemed too sensitive for software protection alone.
