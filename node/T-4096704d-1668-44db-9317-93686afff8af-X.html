Title
NATO Directive on Physical Security
Description

The Directive on Physical Security is published by the NATO Security Committee (AC/35) in support of Enclosure “D” to the NATO Security Policy (C-M(2002)49). This directive contains mandatory provisions and also includes information that clarifies the meaning of those provisions. The directive addresses the following aspects * security requirements; * physical security measures; * minimum standards for the storage of NATO classified information; * protection against technical attacks; and * physical security of communication and information systems (CIS).
