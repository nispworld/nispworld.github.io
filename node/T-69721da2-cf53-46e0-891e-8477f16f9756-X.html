Title
Geographic Data Production Management Application
Description

The Geographic Data Production Management Application enables users to produce authoritative geospatial content by managing and publishing accurate GIS data and cartographic products. This application typically provides a shared work environment for teams of various sizes to improve the quality and value of geospatial data and cartographic products through standardization, repeatability, and configuration of production processes and workflows. Typical functions include * Centralize GIS workflow creation and management to ensure consistency across operations. * Standardized feature collection and validation, cartography, workflows, and data management. * Database development. * Provisioning of consistent review process by automating tasks for spatial data quality control. * Standardized and centralized cartographic production with tools for creating and maintaining derived data, symbology, page elements, and maps.
