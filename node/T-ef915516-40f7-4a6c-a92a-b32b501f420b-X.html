Title
Hardware Security Token Management Application
Description

The Hardware Security Token Management Application enables the user to issue and manage information about issued Hardware Security Tokens. Hardware Security Tokens are physical devices used to enable users physical access to buildings and controlled spaces and/or logical access control in a cyber environment. The most common types of hardware security tokens are smart cards, USB security tokens or one-time password generators. The maintenance of hardware security tokens include issuance, re-issuance, renewal and termination.
