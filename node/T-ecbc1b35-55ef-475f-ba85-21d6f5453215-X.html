Title
Land Execution Process
Description

The Land Execution Process is taking place in the Execution, Assessment and OPLAN Review Processes (Phase 5 of Operations). The purpose of the Land Execution Process is to direct the immediate operations in order to lead the operations. This process is necessary for leading a land operation. This process is concomitant with the Assessment processes and with OPLAN Review processes.
