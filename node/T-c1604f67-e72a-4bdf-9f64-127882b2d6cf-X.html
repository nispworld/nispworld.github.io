Title
Cyberspace Operations Execution
Description

Cyberspace Operations (COs) will be conducted according to the operation plan. As such, Sovereign Cyber Effects Provided Voluntarily by Allies (SCEPVA), when authorised, might give the commander alternative options to create effects. These options are integrated, coordinated and synchronised across force components to achieve objectives as efficiently as possible.
