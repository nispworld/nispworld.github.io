Title
National Prophylaxis Library Process
Description

The National Prophylaxis Library Process describes the activities by a national user to declare the nation’s standard, conditional and other vaccination and chemoprophylaxis practices for an application group (of personnel) against a set of diseases (possibly with constraints of region/AOI, mission or operation). ‘Process Reference ‘ This process is an on-going and routine background activity in support of NRF and operationally-assigned unit prophylaxis readiness. ‘Preconditions ‘ Set of diseases and application groups exist ‘Basic Flow ‘
