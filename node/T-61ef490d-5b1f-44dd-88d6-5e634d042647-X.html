Title
CBRN Algorithm Configuration Process
Description

The CBRN Algorithm Configuration Process exploits the methodology given in AMedP-8. This methodology is based on assumptions and it defines parameters and properties of the casualty estimation process that can be configured. Following this process, the user (with required permissions if applicable) is able to configure a casualty estimation process.
