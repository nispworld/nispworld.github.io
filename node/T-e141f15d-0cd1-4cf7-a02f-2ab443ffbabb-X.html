Title
Email Application
Description

The Email Application provides functionality to exchange electronic mail (email) between users across a digital network. The Email Application typically supports desktop client-based or web-based User Interfaces to allow users to compose and store emails for delivery to one or more recipients, to specify a subject, assign an importance, attach information or to append any other useful information to the email.
