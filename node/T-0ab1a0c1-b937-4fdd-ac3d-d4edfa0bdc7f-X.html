Title
CIS Security Evaluation and Optimization Support Application
Description

The CIS Security Evaluation and Optimization Support Application enables users to evaluate the effectiveness of the actual CIS security during operation. This evaluation is performed through the controlled execution of no-notice, realistic cyber-attacks targeting the CIS, thus evaluating the overall security brought by technical safeguards and implemented processes and procedures. The CIS Security Evaluation and Optimization Support Application also enables the demonstration of the possible impact to businesss processes and mission objectives to senior management, thus allowing them to better prioritize investment in CIS security measures and improve the use of existing resources. Finaly, by exercising processes and procedures in a realistic fashion, the CIS Security Evaluation and Optimization Support Application helps ensure their optimization.
