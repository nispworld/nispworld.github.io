Title
Maritime Targeting Application
Description

NATO uses Joint Targeting process via NATO Joint Targeting System (NJTS). The Maritime Targeting Application receives Targets and displays them. Any action on the actual Targets, such as Destroyed Targets, are updated through NJTS.
