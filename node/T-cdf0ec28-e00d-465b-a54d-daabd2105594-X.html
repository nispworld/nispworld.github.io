Title
MCD Medical Capability Addition Process
Description

The MCD Medical Capability Addition Process describes the activities to add a medical capability (i.e. module) to the Medical Capability Directory (MCD). The purpose of the MCD is to create a catalogue of the medical capabilities of the nations. ‘Preconditions ‘ The member is logged in, which will allow for automatically setting the (parent) nation of any added module. ‘Basic Flow ‘
