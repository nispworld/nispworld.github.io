Title
Word Processing Application
Description

The Word Processing Application enables users to produce (including composition, editing, formatting and possibly printing) any sort of printable material. The application enables users to write articles, books, letter documents, reports, etc. It makes editing a very simple task. The main features of this program are, Font Application, Spell Check, Grammar Check, Thesaurus, Autocorrect, Auto text, Web integration, HTML documentation etc.
