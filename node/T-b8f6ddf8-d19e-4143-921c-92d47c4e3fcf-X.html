Title
ROE Management Application
Description

The Rules of Engagement (ROE) Management Application enables users to develop, manage and disseminate Rules of Engagement. The TOPFAS Working Document ‘ROE Management and Sustaining Use Cases ‘, Version 1.4, Dated 24 June 2016 reviews the ROE Management Process and provides detailed user stories and use cases for the management of ROE and ROE messages during the planning and execution of an operation. Consult this document for an extensive description of the use cases.
