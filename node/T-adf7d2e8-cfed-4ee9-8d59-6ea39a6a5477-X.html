Title
Maritime Incident Recording Application
Description

The Maritime Incident Recording Application provide the user with the capability to record maritime incidents or events such as accidents, detaining, banning, and hijacking. It allows the authorised user to manage the records, including exporting and importing and association of incidents to the vessels in the Vessel Database.
