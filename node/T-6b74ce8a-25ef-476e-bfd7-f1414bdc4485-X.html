Title
Ground Truth Battlespace Objects Services
Description

The Ground Truth Battlespace Objects Services provide the means to discover, exchange, access and modify Ground Truth Battlespace Objects (BSOs) which are generated as input to simulations. The services enable the stimulated development of percieved truth BSOs by Battlespace Object Services.
