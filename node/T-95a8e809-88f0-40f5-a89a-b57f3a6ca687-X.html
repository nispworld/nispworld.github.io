Title
Maritime Simulation Management Application
Description

The Maritime Simulation Management Application enables the user to use the Maritime Operation Management for training and exercise purposes and create a Training Environment. The Training Systems use simulators to simulate external data sources according to a common reference, ‘Ground Truth ‘ within the Training Environment. It is configurable to use the available data simulators and to provide coordinated data inputs to the simulators (i.e. Object Data).
