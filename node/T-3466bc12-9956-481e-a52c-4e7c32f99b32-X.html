Title
Host Nation Support Application
Description

The Host Nation Support (HNS) Application enables operators to manage all aspects of Host Nation Support in support of Alliance logistics capabilities and shortfalls regarding maintenance, transportation, medical support, supply, communications, force protection, billeting and messing, facilities and infrastructure. It allows maintaining a HNS Catalogue and provides the required HNS information to the other functional services. The Deferred Host Nation Support Catalogue will contribute to this functionality.
