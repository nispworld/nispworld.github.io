Title
JISR Single Source Exploitation Applications
Description

The Joint Intelligence, Surveillance and Reconnaissance (JISR) Single Source Exploitation Applications enable users to review, annotate, tag, categorize, transform, and manipulate ISR data obtained from a single type of sensors in order to extract useful and valid intelligence out of the received data and generate relevant reports for dissemination.
