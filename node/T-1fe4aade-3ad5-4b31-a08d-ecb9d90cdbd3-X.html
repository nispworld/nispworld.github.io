Title
Relational Database Services
Description

The Relational Database Services provide controlled access to a collection of data items organized as a set of formally-described tables from which data can be accessed or reassembled in many different ways without having to reorganize the database tables. Relational Database Services can be accessed through the Structured Query Language (SQL). SQL statements are used for schema manipulation, data manipulation and information retrieval.
