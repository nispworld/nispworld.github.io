Title
Aeromedical Evacuation Application
Description

The Aeromedical Evacuation Application enables users to make decisions for patient movement operations, matching patient requirements with medical treatment capability and available lift. The application assists users in forecasting, planning, and conducting reactive planning for operations ranging from single patient to mass casualty events. During execution, it provides in-transit visibility and replanning capability.
