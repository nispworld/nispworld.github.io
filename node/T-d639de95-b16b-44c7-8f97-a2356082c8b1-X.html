Title
Request Processing Application
Description

The Request Processing Application enables users to process customer requisitions for stocked and non-stocked items under the direct exchange procedure. The Request Processing Application covers the recording of customer requisitions, the release from stock, the generation of shipping documents and finally the confirmation of the physical release of the material.
