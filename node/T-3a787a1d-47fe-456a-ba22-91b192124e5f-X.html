Title
Manpack Equipment
Description

Manpack Equipment is a type of user equipment that is battery powered and that can be used out in the field, away from recharge facilities for a length of time. This is vital to the success of many mission. Typically manpack equipment is a compromise between weight (~ 4-5 kg), volume, level of ruggedization and portability. Often this equipment includes removable control units that allow using the device when on the move.
