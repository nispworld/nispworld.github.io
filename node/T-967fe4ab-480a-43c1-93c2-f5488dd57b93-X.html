Title
Information Labeling Application
Description

The Information Labeling Application enables users to select metadata that applies to an information object for the purpose of creating a label, and mapping labels of ‘foreign ‘ security policy. It allows to facilitate the determination of the protection requirement for the information object, the release of the information object, or the determination of the mission value of the information object, as captured by release and protection policies defined for each metadata entry.
