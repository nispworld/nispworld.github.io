Title
NATO Emitter Database Application
Description

The NATO Emitter Database Application enables users to maintain entities in the NATO Emitter Database (NEDB). The application must be available to be updated by nations during crisis and provided to designated SC, JFC and component HQs for the planning and conduct of operations. The application will support EW processing, including automated correlation and direction finding. The application will allow the user to visualize the emitter information in textual or geographic views, including emitter coverage, threat rings, lines of bearing, etc.
