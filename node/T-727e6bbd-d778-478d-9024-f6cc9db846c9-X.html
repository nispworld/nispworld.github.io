Title
Desktop Communications
Description

Desktop Communications is a type of user equipment that provides various telepresence, video, and voice communications solutions. This equipment includes traditional analogue voice telephones, circuit switched Video Teleconference systems, as well as modern media phones, combining a personal video conferencing system with voice over IP (VoIP) telephone and microbrowser for real-time delivery of personalised Web content. This equipment can be integrated with key business applications via open application programming interfaces (API).
