Title
NATO Crisis Response System Application
Description

The NATO Crisis Response System (NCRS) Application enables the user to manage and respond to crises, including sudden shifts in the security environment, on the basis of a comprehensive set of options and measures, taking full advantage of the tools and capabilities available to NATO. The application also enables the user to manage a repository of crisis response messages, drafting and issuing crisis response messages, and tracking these messages.
