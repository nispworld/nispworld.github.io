Title
Remote Video Terminal
Description

A Remote Video Terminal (RVT) is user equipment that is man-portable and displays sensor data from multiple airborne platforms. This eqipment typically supports multi-band digital and analog signals. Some RVTs include the capability to interpret metadata, transmitted on many Unmanned Aerial Sensor (UAS) platforms. RVTs systems are typically designed around ruggedized laptops.
