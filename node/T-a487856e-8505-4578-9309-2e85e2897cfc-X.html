Title
Tape Device
Description

A Tape Drive is a data storage device that provides sequential access on a magnetic tape. Tape media generally has a favorable unit cost and a long archival stability and is typically used for offline, archival data storage. A tape drive must physically wind tape between reels to read any one particular piece of data, as a result, tape drives have very slow average seek times to data. (Source Wikipedia)
