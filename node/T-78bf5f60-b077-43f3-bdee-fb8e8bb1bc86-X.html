Title
Interference Analysis Application
Description

The Interference Analysis Application enables users to perform interference analysis considering equipment, locations, terrain, antenna and frequency-dependent coupling; to predict interference sources and levels for existing or proposed spectrum use; and to visualisation spectrum analysis results (spectrum occupancy and record conflict charts).
